Password Strength & Breach Checker

A simple Java-based application that checks the strength of passwords and verifies if they have been exposed in known data breaches. Ideal for practicing cybersecurity principles and Java file handling.


Features

Password Strength Checker: Evaluates passwords for length, complexity, and use of special characters.

Breach Checker: Compares passwords against a list of known compromised passwords.

File Handling: Reads from and writes to text files to store passwords and results.

Console-based: Lightweight and easy to run without a GUI (JavaFX optional).

Technologies Used

Java (core language)

Java File I/O (for storing and reading password data)
